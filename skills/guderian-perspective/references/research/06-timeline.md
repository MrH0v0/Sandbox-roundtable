# 06 - 人物时间线

| 时间 | 事件 | 对思维框架的影响 |
|------|------|----------------|
| 1888 | 出生于 Kulm，普鲁士军人家庭 | 普鲁士军官传统、纪律和职业军人身份是底层背景 |
| 1908 | 成为轻步兵军官 | 进入传统军队体系 |
| 1914-1918 | 一战中主要任无线电通信和情报岗位 | 形成对通信、情报和指挥链的敏感；并非亲历坦克攻击后顿悟 |
| 1919 | 短期参与东线/波罗的海相关军事行动 | 早期反布尔什维克和东线经验背景 |
| 1922 | 转入摩托化部队相关岗位 | 开始长期下注机动、运输和装甲化 |
| 1920s-1930s | Reichswehr 秘密研究、演习和吸收国际装甲理论 | 形成“技术 + 组织 + 训练 + 通信”的系统视角 |
| 1935 | 德国建立首批 Panzer divisions | 理论获得组织载体 |
| 1937 | 出版 `Achtung - Panzer!` | 将兵种理论、公关和组织游说结合 |
| 1939 | 指挥 XIX Army Corps 参加波兰战役 | 验证快速装甲突击和合围思路 |
| 1940 | Sedan 突破并向英吉利海峡推进 | 速度、集中、抗命式主动性达到顶峰 |
| 1941 | 指挥 Panzergruppe 2 参加 Barbarossa | 模型遭遇补给、冬季、敌方韧性和战略目标冲突 |
| 1941-12 | 因撤退争议被 Hitler 解职 | 暴露主动性与服从体系的冲突 |
| 1943-03 | 任 Inspector General of Armoured Troops | 转向训练、生产优先级和装甲兵恢复 |
| 1944-07 | 7月20日密谋后任陆军总参谋长 | 技术专家进入最高战略和政治绞盘，局限暴露 |
| 1945-03 | 再度被 Hitler 解职 | 通过升级争执脱离沉船位置 |
| 1951/1952 | 出版回忆录 `Panzer Leader` | 强化个人神话，也参与战后德军自我辩护叙事 |
| 1954 | 去世 | 历史形象长期被“装甲先知”和后来的批判性研究拉扯 |
